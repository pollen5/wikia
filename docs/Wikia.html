search (query, options)

Do search for a given phrase. (only available if a wiki is provided on the instance.)
Name Type Description
query String Search query.
options Object optional Options for this search.
Name Type Default Description
type String articles optional The search type, either articles (default) or videos. For 'videos' value, this parameter should be used with namespaces parameter (namespaces needs to be set to 6)
rank String optional The ranking to use in fetching the list of results, one of default, newest, oldest, recently-modified, stable, most-viewed, freshest, stalest
limit Number optional Limit the number of results.
minArticleQuality Number optional Minimal value of article quality. Ranges from 0 to 99.
batch Number optional The batch (page) of results to fetch.
namespaces String optional Page namespace number, see more: http://community.wikia.com/wiki/Help:Namespaces
